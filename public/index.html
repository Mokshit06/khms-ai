A new way for your devices to be helpful

Live Captions generates closed captions for audio and video elements, right on the user's device. This provides a much better user experience for users who have hearing impairment, as this can help them know what is being said in the video. This would work similar to platforms like Youtube which generate captions but works for every video without needing to upload it anywhere. This can also be made into a Browser extension which would automatically add captions for every video.

How does it work?

Live Captions converts the URL given by user to a Buffer using Node.js, converts the video to an audio using ffmpeg and then uses Google Cloud Speech-To-Text API for generating closed captions over that Buffer. It then takes the output of Speech-To-Text API and then converts it to a format called WebVTT which browsers understand. This way it can generate live captions really efficiently and quickly.
